Elektronikon Touch Access Level Quick Guide

Applies to compressors equipped with the Elektronikon Touch controller. Use during normal operation and fault recovery.

Back to Guides

Purpose

  • Enter Advanced User mode while the unit is in Standby
  • Exit the mode safely after completing the task

Video. Entering Advanced User Mode

Access Levels Overview

  • Basic. Default operator interface
  • Advanced. Restricted operational functions
  • Service. Manufacturer or maintenance access only

Entering Advanced User Mode

  1. Start from the Home screen
  2. Tap the user icon in the top corner
  3. Select the Advanced user level
  4. Confirm the selection
Validation
  • User level indicator shows Advanced with a star symbol
  • Additional menus are visible

Rules While in Advanced Mode

  • Do not modify control parameters
  • Do not access Service menus
  • Follow written procedures only
Advanced mode provides expanded control access. Incorrect changes lead to equipment trips or damage.
