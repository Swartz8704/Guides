Elektronikon Touch Access Level Quick Guide

Back to Guides

Applies to compressors using the Elektronikon Touch controller. Use this guide during normal operation and fault recovery.

Purpose

  • Enter Advanced User mode (Standby Mode )
  • Exit safely after task completion

Video. Entering Advanced User Mode

Access Levels Overview

  • Basic. Default operator view
  • Advanced. Protected operational functions
  • Service. OEM or maintenance only

Entering Advanced User Mode

  1. Start at the Home screen
  2. Tap User in top corner
  3. Select User level Advanced
  4. Confirm the selection
Validation
  • User level indicator shows Advanced ( * )
  • Additional menus are visible

Rules While in Advanced Mode

  • Do not change control parameters
  • Do not access Service menus
  • Follow written procedures only
Advanced mode provides control access. Incorrect changes cause trips or damage.
